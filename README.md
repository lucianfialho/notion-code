# notion-code

Claude Code for Notion — manage your entire Notion workspace from the terminal using natural language.

## Install

```bash
npm install -g notion-code
```

## Quick Start

```bash
# Authenticate with Notion
notion-code auth login

# Start interactive mode
notion-code

# Or run a single command
notion-code -p "show me all tasks due this week"
```

## Requirements

- Node.js >= 18
- [Claude Code](https://claude.ai/code) subscription (for the Agent SDK)
- Notion account
